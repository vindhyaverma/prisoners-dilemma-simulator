# Prisoner’s Dilemma Strategy Simulator

Interactive Prisoner’s Dilemma Strategy Simulator built using Python, Gradio, and Google Colab.

## Features
- Multiple AI strategies
- Round-by-round simulation
- Final score calculation
- Interactive Gradio GUI
- Adjustable number of rounds

## Technologies Used
- Python
- Gradio
- Google Colab

## Strategies
- Always Cooperate
- Always Betray
- Random

## Objective
Understand cooperation, betrayal, and strategic decision-making using Game Theory concepts.
